# Coldstorm AI — Investigation Unit Economics
## March 7, 2026
## Purpose: Decide whether the multi-agent investigation business is financially worth doing repeatedly

## Bottom Line

Yes. The economics are attractive enough to perform regularly if Coldstorm sells the work as a finished intelligence product, not as cheap "AI help."

The reason is simple:
- market pricing for investigation work is still anchored to human-delivered boutiques
- Coldstorm's delivery cost is much closer to software + analyst review
- the spread between price and cost is wide enough to support high-margin recurring work

The real requirement is discipline:
- standardized workflows
- strict QA
- clear scope boundaries
- pricing that reflects client value, not API cost

## What "Cost to Serve" Includes

Cost to serve is not just API spend. Use this model:

| Cost Bucket | Typical Range / Matter | Notes |
|---|---:|---|
| LLM / agent runtime | $20-$150 | Depends on depth, retries, report length |
| External data / API calls | $20-$250 | OpenCorporates, sanctions, OSINT sources, etc. |
| Human review / QA time | $100-$800 | This is where most variable cost lives |
| Delivery formatting / revisions | $10-$200 | Depends on polish expectations |
| Total variable cost | $150-$1,200+ | Varies by matter complexity |

The trap is ignoring review time. If review is uncontrolled, the margins collapse.

## Why This Can Work Repeatedly

The business is worth doing repeatedly if Coldstorm preserves four things:

### 1. Repeatable Workflow
- same collection structure
- same verification layer
- same contradiction-finding logic
- same final review checklist

This repo already shows the core IP for that in the Swiss Fox pipeline and the litigation multi-channel evidence work.

### 2. Review Discipline
- AI does not remove human responsibility
- AI removes low-value analyst labor
- review must be limited, structured, and senior

If every report turns into custom hand-editing for hours, the economics degrade fast.

### 3. Proper Positioning

Sell:
- decision-ready intelligence
- faster turnaround
- broader source coverage
- audit trail
- expert-reviewed output

Do not sell:
- "cheap AI report"
- raw model output
- generic research help

### 4. White-Label Channels

The highest-leverage path is not direct retail.

The best near-term path is:
- investigation boutiques
- compliance consultancies
- law firms
- specialist advisory firms

Why:
- they already have clients
- they understand the value
- they can resell the work
- they create retainer opportunities faster than direct acquisition

## Main Risks to the Economics

| Risk | Impact | Mitigation |
|---|---|---|
| Underpricing because "AI should be cheap" | Destroys margin | Anchor to boutique investigation pricing |
| Too much bespoke work | Kills repeatability | Productize 3-4 standard engagement types |
| Weak quality control | Refunds, trust loss | Mandatory human review and source-linked output |
| Expensive proprietary APIs too early | Margin compression | Use open + targeted paid sources, not enterprise stacks first |
| Founder bottleneck | Limits scale | Standardize review, templates, and escalation rules |

## Recommended Commercial Policy

### Minimum viable pricing floor
- OSINT: never below $1,500
- Standard EDD: never below $2,000
- Enhanced EDD: never below $3,500
- Retainer: never below $6,000 / month

### Discounting rule
- no unconditional discounting
- discounts only in exchange for one of:
  - recurring volume
  - white-label partnership
  - faster payment
  - reduced scope

### Packaging rule
- sell three clear tiers
- define what is in scope
- define revision limits
- define turnaround class

## Decision

The multi-agent investigation business has good enough financial reward to perform regularly.

Not because the models are cheap.

Because the market still pays human-investigation prices for outcomes, while Coldstorm can deliver those outcomes with a much lower cost structure if the workflow remains disciplined and expert-reviewed.
