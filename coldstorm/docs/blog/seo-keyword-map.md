# SEO Keyword Map — Coldstorm AI (Litigation Intelligence Engine)

*Last updated: March 4, 2026*

---

## Strategy Overview

**Primary goal:** Rank for buyer-intent keywords that mid-market law firm partners and corporate counsel search when evaluating AI litigation tools.

**GEO-first approach:** All content is structured for AI agent discoverability (ChatGPT, Perplexity, Google AI Overviews) — not just traditional Google ranking. This means:
- Answer-first content structure (direct answer in first 40-60 words)
- Headers written as natural language questions
- FAQ sections embedded for schema markup
- Source-quality citations and factual claims throughout

**Competitive positioning:** Avoid head-to-head competition with Harvey/CoCounsel on high-volume keywords ("legal AI", "AI for lawyers"). Instead, dominate low-competition, high-intent keywords around multi-channel evidence analysis and contradiction detection — capabilities no competitor offers.

## Technical SEO Recommendations

### robots.txt Additions
```
User-agent: *
Allow: /blog/
Allow: /llms.txt
Sitemap: https://coldstorm.org/sitemap.xml
```

### Schema Markup (Add to Each Blog Post)
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "[Blog Title]",
  "author": {
    "@type": "Organization",
    "name": "Coldstorm AI"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Coldstorm AI",
    "url": "https://coldstorm.org"
  },
  "datePublished": "2026-03-04",
  "dateModified": "2026-03-04",
  "description": "[Meta description]",
  "mainEntityOfPage": "[Blog URL]"
}
```

### FAQ Schema (Add to Each Blog Post's FAQ Section)
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "[Question text]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "[Answer text]"
      }
    }
  ]
}
```
